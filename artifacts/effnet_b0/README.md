# EfficientNet-B0 Artifact

This folder contains the Kaggle-trained EfficientNet-B0 baseline artifact generated by `notebooks/02_effnet_b0_baseline.ipynb`.

## Files

- `best_effnet_b0.pt`: best checkpoint by validation accuracy.
- `labels.json`: index-to-label mapping for the 206 training classes.
- `history.csv`: epoch-level training history.
- `train_folds.csv`: metadata with fold assignments used for training and validation.

## Training Summary

- Epochs: **5**
- Best validation accuracy: **0.7318**
- Final validation loss: **1.6846**
- Backbone: **EfficientNet-B0**
- Input: **5-second mono mel-spectrograms**

Use `notebooks/04_effnet_b0_submission.ipynb` to generate a Kaggle `submission.csv` from this artifact.
