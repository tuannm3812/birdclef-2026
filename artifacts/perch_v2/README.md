# Perch v2 Probe Artifact

This folder contains the Kaggle-trained probe head for frozen Google Perch v2 embeddings, generated by `notebooks/03_perch_v2_probe.ipynb`.

## Files

- `best_perch_probe.pt`: best shallow PyTorch probe checkpoint by validation accuracy.
- `labels.json`: index-to-label mapping for the 206 training classes.
- `history.csv`: epoch-level probe training history.

The full `train_embeddings.npz` file was inspected but not committed because it is about **202 MB**. Keep it as a Kaggle output artifact or store it with Git LFS/release assets if needed.

## Embedding Summary

- Embedding shape: **35,549 x 1,536**
- Embedding dtype: **float32**
- Labels: **206 classes**
- Best validation accuracy: **0.8403**

## Notes

The probe converges quickly and outperforms the EfficientNet-B0 baseline validation accuracy by about **10.9 percentage points**. The validation curve is fairly flat after epoch 1 while train loss keeps decreasing, which suggests the linear/nonlinear probe is already extracting most of the available signal from frozen Perch embeddings and may start overfitting with longer training.
